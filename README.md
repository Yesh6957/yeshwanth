# Yeshwanth — Portfolio

Lightweight Vite + React + TypeScript portfolio showcasing projects, photography and contact details.

## Features
- Vite + TypeScript frontend
- Tailwind CSS design system
- Portfolio sections: Hero, About, Projects, Photography, Skills, Contact, etc.
- Continuous marquee photography component
- Resume download button (PDF served from src



## Project structure (high-level)
- src/ — application source
  - components/portfolio — sections (Hero, Projects, Photography, Contact, etc.)
  - main.tsx, App.tsx, index.css
- public/ — static assets (images, favicon)
- package.json — scripts and deps
